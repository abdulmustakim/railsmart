# RailSmart SmartCoach Finder - Static Version

This version uses only:

- HTML
- CSS
- JavaScript

No Node.js, no backend server, no database, no API key.

## How to run

Open `index.html` directly in any modern browser.

## How to host online for free

### GitHub Pages
1. Create a GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and this README file.
3. Go to Repository Settings > Pages.
4. Select branch: `main` and folder: `/root`.
5. Save. Your site will be live as a GitHub Pages link.

### Netlify
1. Open Netlify.
2. Drag and drop the whole `railsmart-static` folder.
3. Netlify will instantly publish the website.

## Features included

- Confirmed-seat-only train search
- Flexible date finder
- Nearby station suggestions
- Coach crowd level
- Approximate human count by coach
- Phone signal comparison: Jio, Airtel, Vi, BSNL
- Best coach recommendation
- Seat alert saved in browser localStorage
- User coach report saved in browser localStorage
- Dashboard

## Important note

This static version uses demo/sample data. Official real-time railway seat availability requires authorized railway/IRCTC data access. A pure HTML/CSS/JS static site cannot safely provide official live railway data by itself.
